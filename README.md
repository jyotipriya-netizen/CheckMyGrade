# CheckMyGrade Application

CheckMyGrade is a Python-based student management system that allows users to manage student records and analyze grades using a menu-driven interface

# 1. Display Students
View all student records including email, name, course, and marks.

# 2. Add Student
Add a new student by entering:
- Email
- First Name
- Last Name
- Course ID
- Marks

# 3. Delete Student
Delete a student record using email.

# 4. Update Student
Update student details such as:
- First Name
- Last Name
- Course ID
- Marks

# 5. Search Grade by Email
Retrieve the grade of a student using their email.

# 6. Search Marks by Email
Retrieve marks of a student using their email.

# 7. Sort by Marks
- Ascending
- Descending

# 8. Sort by Email
Display students sorted alphabetically by email.

# 9. Show Average Marks
Calculate and display average marks of all students.

# 10. Show Median Marks
Calculate and display median marks.

# Technologies Used
- Python
- CSV Files
- VS Code
